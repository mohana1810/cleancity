# 🌿 CleanCity - Waste Management System

CleanCity is a full-stack web application that allows users to report waste issues in their area and helps authorities manage and track waste collection efficiently.

## 🚀 Features

* 📍 Report waste issues with location and images
* ♻️ Categorize waste (General, Recyclable, Hazardous, Organic)
* 👤 User authentication (Citizen, Driver, Admin)
* 📊 Dashboard for tracking reports
* 🏆 Leaderboard with points system
* 📸 Image upload using Cloudinary

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Cloud Storage: Cloudinary

## 📂 Project Structure

backend/
frontend/

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies

npm install

3. Create `.env` file and add:

PORT=5000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret

CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret

4. Run backend:
   npm start

5. Run frontend:
   npm start
